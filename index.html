Retrieval-Augmented Generation (RAG) using Mistral & FAISS

This project demonstrates a complete Retrieval-Augmented Generation (RAG) pipeline implemented locally using LangChain, FAISS, and the Mistral large language model via Ollama.The pipeline retrieves contextually relevant chunks from a document (like a research paper or PDF),augments the user’s query with that context, and generates grounded answers using the local LLM.
